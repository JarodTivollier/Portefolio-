SAE4.01

Correction du code d'un site web et de son application mobile

Objectif


L’Objectif de cette SAE4.01 était de corriger le code d'un site web de suivi de stage et de son application mobile.


Résumé

Pour cette SAE, le site web était destiné aux enseignants et l'application mobile pour les étudiants. Le code était mal structuré, le nom des variables n'était pas pertinent, beaucoup de redondance au niveau des fonctions et des fonctions incomplètes.

Ma partie et difficultées


Lors de cette SAE j'ai était responsable de l'application mobile je me suis donc principalement occupé de corriger le code et le remanier avec l'aide de 2 camarades qui ce sont occupé des interfaces de l'application mobile

voici le lien du rapport de ce projet "Rapport"

Lors de cette SAE, j'ai rencontré quelques difficultés principalement le fonctionnement des fonctions étant donné que certaines n'étaient pas finies et qu'il n'y avait pas de commentaires. Comprendre à quoi elles servent a été ma principale difficulté lors de ce projet.


Compétences acquise


Durant cette SAE j'ai acquis diverse compétences telle que par rapport au BUT, la compétence UE1 pour le développement mobile et web, par rapport au niveau personnel, cela m’a apporté plus de connaissance sur le langage de programmation Java sur Android et amélioré ma logique de compréhension.
